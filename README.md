# Event-management-system
A Flask-based web application with user and admin roles for event registration, management, and tracking.


# Overview
This is a web-based Event Management System built using Flask that allows users to register for events and enables admins to manage event data efficiently.
The system includes authentication, event registration, and an admin dashboard for monitoring all activities.

# Features
# User Features
* User login system
* Register for events
* View registered events
* Submit additional information (phone, college, message)

# Admin Features
* Admin login
* View all event registrations
* Delete event entries
* View additional user details

# Tech Stack
* Python
* Flask
* HTML (via `render_template_string`)
* CSS (embedded styling)

# How It Works
* Users log in and register for events
* Event data is stored temporarily in Python lists
* Admin can view and manage all registrations
* Additional user details are collected through forms
